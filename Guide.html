Initial setup and configuration

Setup

  1. Download Nodejs v16.13.0 from Node.js — Node v16.13.0 (LTS) (nodejs.org)
  2. Install it.
  3. Open command prompt, goto project folder.
  4. Run command yarn, press enter.

Configuration

  1. Paste your token image in project directory and name it as image.png
  2. Open config.js
  3. Values for variables:

    • PRIVATE_KEY: Replace aaaaa with your wallet private key.

    • endpoint: Replace aaaaa with your RPC HTTP(S) URL.

    • NFT_STORAGE_TOKEN: Goto https://nft.storage/. Then sign up, you can use temp mail. Then goto Manage API Keys - NFT Storage and click new key. It will generate a key for you. Copy it and aaaaa with your nft.storage api key. Note: Nft.storage is free storage for tokens metadata.

    • revokeMintBool: If you dont want to revoke mint, set it to false. By default it has been set to true.

    • revokeFreezeBool: If you want to revoke Freeze, set it to true. By default it has been set to false.

    • tokenInfo:
      Example: Cant add in guide because formatting will destroy your understanding just open config.js and read it there.

      • amount: Set amount of tokens you want to create.
      • decimals: Set decimals of token.
      • metadata: Leave it empty as it is.
      • symbol: Set token symbol. (Note: It should not be longer than 10 Characters)
      • tokenName: Set token name

    • metaDataforToken:
      Example: Cant add in guide because formatting will destroy your understanding just open config.js and read it there. Each of these sections in metadata are optional. If you dont want to add them, you can remove them but i have added enough information for your understanding incase you wanted to add every piece in future.

      1. image: Leave it empty as it is.
      2. description: Add description of your token.
      3. extensions:
        • website: Replace it with your website url.
        • twitter: Replace it with your twitter url as shown in example in config.js.
        • telegram: Replace it with your telegram url.
        • You can add further socials also like discord as "discord": "url_to_discord"
      4. tags: Enter your tags as they are defined in example.
      5. Creator:
        • name: Ever seen tokens created by dex ui labs? Enter something like Papi chulo as the creator. Upto u whatever u wanna add.
        • site: Site of the creator. e.g. when u create tokens with dexui, dex ui adds their website here. So people can come and create through them :D. You can add my github lol.

We done with configuration.

How to Run the project.

Just run the command node ./main.js and it will create the token for you if everything you entered was correct :D.
